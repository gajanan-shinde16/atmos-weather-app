# Atmos Weather App

A modern, responsive weather application built with React and Vite, utilizing Material UI for the interface and OpenWeatherMap API for real-time data.

## Features
- Search weather by city name.
- View current temperature, humidity, and daily highs/lows.
- Clean, responsive UI using Material UI components.
